# What is docs.skoolapi.com? Is SkoolAPI a third-party service that provides API access to Skool.com data? What endpoints does it offer, what does it cost, and how reliable is it? Is

## Key findings

**docs.skoolapi.com is the documentation site for SkoolAPI, a third-party service that provides unofficial programmatic access to Skool.com's core functionality, such as sessions and webhooks, to enable integrations and automation for developers and community managers.[1][2][5]**

SkoolAPI is a **third-party service**, not officially affiliated with Skool.com, designed to address limitations in Skool's native features by offering API endpoints for features like **Sessions** (authenticated connections to Skool accounts) and **Webhooks**.[1][2][5] It requires users to generate an API key via a SkoolAPI dashboard, which is included in request headers (e.g., `X-Api-Secret: your_api_secret_here`) for authentication, granting full access to the linked Skool account.[1]
