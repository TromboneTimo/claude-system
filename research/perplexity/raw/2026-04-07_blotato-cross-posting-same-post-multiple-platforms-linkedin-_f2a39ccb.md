# Blotato cross-posting same post multiple platforms LinkedIn Twitter Facebook Instagram. When using blotato_create_post to cross-post to multiple platforms simultaneously, does Blot

## Key findings

**Blotato does not automatically handle platform differences like character limits, hashtag formatting, or image requirements in a single `blotato_create_post` call for cross-posting.** Instead, the standard workflow requires **separate `blotato_create_post` calls per platform** (or per account), as each platform has unique specs that must be addressed individually.[1][2][3][9]
