# Blotato Remix Feature: What's Actually Available

Based on current documentation, here's what Blotato's remix feature actually does:

## Core Functionality

The **remix feature transforms existing content into adapted versions across multiple formats and platforms**[1]. The basic workflow is: click "Remix" on a post → Blotato generates an adapted version → you add personal touches and customizations[1].

## Input Content Types

Blotato can remix the following source materials:

- YouTube videos[3]
- TikTok videos[3]
- Articles[3]
- PDFs[3]
- Podcasts[3]
- Newsletters[3]
- Existing social posts[1]

If you lack industry-specific sources, you can input any topic via the Perplexity tab, and Blotato will research and generate fresh content to remix[1].

## Output Formats & Transformations

Documented transformation examples include:

- 1 TikTok → Instagram post + tweet[3]
- 2 articles + 2 PDFs → viral YouTube script[3]
- 3 YouTube videos → 1500-word blog post + LinkedIn post + tweet[3]

The platform supports **cross-posting to LinkedIn, TikTok, X (Twitter), Facebook, Instagram, Threads, and Pinterest**[4].
