# is Claude Code scheduled agents API down right now April 2026? any outages reported?

## Key findings

**No, the Claude Code Scheduled Agents API is not down right now on April 4, 2026.** The official Claude status page reports no incidents today, with the most recent issues (elevated errors on Claude Sonnet 4.6 on April 3 and various errors/timeouts on April 1) all resolved[3]. Past outages include a major Claude disruption on March 2, 2026, affecting the web app, developer console, and some API methods (resolved by March 2 evening), but the Claude API was reported operational during much of it[1].

Recent reports mention unrelated bugs like scheduled cron jobs timing out after a 2026.4.1 upgrade[5] and main agents lacking tool access in scheduled tasks[6], but these are software issues, not service outages. No current downtime is noted for the Scheduled Agents API specifically[2][3]. For real-time checks, monitor status.claude.com or implement API health scripts[2].
