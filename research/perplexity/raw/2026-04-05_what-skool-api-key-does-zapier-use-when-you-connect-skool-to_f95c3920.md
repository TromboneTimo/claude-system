# What Skool API key does Zapier use? When you connect Skool to Zapier, what triggers and actions are available? What data fields are exposed? Does Skool provide an API key in the ad

## Key findings

Skool does not use a predefined or universal **API key** in Zapier; instead, each Skool group admin generates a unique **group-specific API key** from their Skool admin settings, which is then entered into Zapier for authentication along with the group's URL (the part after "skool.com/").[1][2][5][6] This key is provided directly in Skool's admin panel under Plugins > Zapier integration (toggle on, copy key), and it can be used outside Zapier for custom integrations, though refreshing it breaks existing Zaps until updated.[2][4][6]
