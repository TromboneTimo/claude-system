## Installation Steps

**Install Hermes Agent:** You can install Hermes Agent with a single command, which automatically creates a virtual environment containing all required Python and Node.js dependencies.[2]

**Install Ollama:** Download and install Ollama from ollama.ai, which manages local AI models on your device and integrates with Hermes Agent.[3]

**Download a Local Model:** Pull a model into Ollama. Recommended options include:
- **Qwen3.5-9B** (approximately 5 GB on disk, requires ~10-12 GB RAM)[4]
- **GPT Microsoft's 20B parameter model**[2]
- **Gemma 4**[3]
- **GLM 4.7 Flash** (requires 25 GB VRAM)[1]

## Mac-Specific Requirements

For optimal performance on Mac, use **Apple Silicon (M1 and later)**, which provides GPU acceleration.[4] Intel Macs will work with llama.cpp but without GPU acceleration, resulting in significantly slower performance.[4]
