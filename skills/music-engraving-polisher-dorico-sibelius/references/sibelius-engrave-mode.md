# Sibelius Engraving Quick Reference

## Where the rules live

- **House Style > Engraving Rules** (score-level rules)
- **House Style > Edit All Fonts / Edit Text Styles** (text appearance)

## Common adjustments

### Note spacing
- House Style > Engraving Rules > Note Spacing
- Minimum space: 1.2 (moderate) / 1.4 (strict)

### Dynamics
- Distance from staff: 3.5sp
- Minimum gap from other objects: 0.5 (moderate) / 1.0 (strict)

### Staves
- Default gap above first staff and between staves: 7 (moderate) / 8 (strict)

### Beams
- Default stem length: 3.5sp
- Maximum beam slope: 3sp

## Re-import

File > Open > `polished.musicxml`. Sibelius asks about import options; accept defaults.
Preserved: notes, rhythms, articulations, dynamics.
Not preserved: hand-placed objects from original unless exported with overrides.
