# Style Preset Comparison

| Setting | conservative | moderate (DEFAULT) | strict |
|--------|--------------|-------------------|--------|
| Min note distance (sp) | 1.1 | 1.2 | 1.4 |
| Staff distance (sp) | 6.5 | 7 | 8 |
| Min system distance (sp) | 8.5 | 9 | 10 |
| Dynamics min gap (sp) | 0.3 | 0.5 | 1.0 |
| Articulation min gap (sp) | 0.3 | 0.5 | 0.75 |
| Accidental column gap (sp) | 0.22 | 0.3 | 0.35 |
| Rehearsal mark padding (sp) | 0.4 | 0.5 | 0.75 |
| Measure spacing factor | 1.3 | 1.5 | 1.75 |

## When to pick each

- **conservative**: User has hand-tweaked layout extensively in Dorico/Sibelius and just wants collision floors enforced. Preserves as much of the original as possible.
- **moderate (DEFAULT)**: 95% of cases. Dorico-ish defaults with tightened spacing. Good for final parts, concert programs, and intermediate drafts.
- **strict**: Behind Bars / publisher-grade. Most generous spacing, aggressive re-flow, zero tolerance for collisions. Use for final publisher-bound output or professional commissioning.
