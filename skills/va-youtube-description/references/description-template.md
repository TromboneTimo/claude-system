## Voice fingerprint (carry into every description)

1. **Direct address.** "You've spent...", "You still can't...", "Your setup is."
2. **Dollar anchors.** Use specific money figures from the video. $500 / $1,000 / $3,000 is real and concrete. Generic "a lot of money" is dead.
3. **Flip the source of the problem.** Every video pivots from "the external thing they blame" to "their face / setup / apparatus". This flip is the entire pitch.
4. **Promise a 30-second test.** Harrison's content always gives one self-diagnostic move the viewer can do RIGHT NOW. Name it in paragraph 2.
5. **The reframe sentence.** Paragraph 3 is the comment-bait. It crystallizes the insight. Always include it.
6. **No fluff phrases.** Cut: "in this comprehensive guide", "we'll dive deep into", "everything you need to know about", "let's explore", "in today's video".
7. **No teacher-speak.** Cut: "fundamentals", "the basics", "key concepts", "core principles". Harrison talks about apparatus, setup, embouchure, placement, pressure, the face. Concrete body parts beat abstract pedagogy.

## Curiosity-chapter rules

Apply ALL of these:

1. **No bare nouns as titles.** "Demonstration" is dead. "Wayne Bergeron mouthpiece: locked-in vs misaligned embouchure" is alive.
2. **Title is a mini-hook.** Read each title alone. Does it pull a reader who hasn't seen the video?
3. **Concrete proper nouns help.** Wayne Bergeron, Vertical Alignment System, pistol-grip, $3,000. Specific beats general.
4. **Two-clause titles work.** "The truth: [insight]". "The lie: [external claim]". "Why I built [system]". "The correct order: [sequence]".
5. **The CTA chapter is always last.** Title: "Free masterclass + what to watch next" (or near-variant). Don't title it "Outro" or "Conclusion".
6. **Each chapter at least 10 seconds.** YouTube enforces this. If two consecutive beats are <10s apart, merge them.
7. **First chapter MUST be at 0:00.** YouTube enforces this.

## Hashtag rules

- Default set: `#trumpetlife #trumpet #trumpetgear`
- Topic add-ons (max 1 extra): `#embouchure` (embouchure videos), `#highnotes` (range/upper register videos), `#trumpetteacher` (pedagogy critique videos)
- Hard bans: `#music`, `#brass`, `#tutorial`, `#youtube`, `#shorts`, `#viral`, `#fyp`, generic genre tags
- Max total: 4

## Things this template does NOT need

- Emoji bullets in the description body (only the YouTube playlist link uses the bullet glyph, and that's auto-rendered)
- "Subscribe!" CTAs in the description (handled by end screen + thumbnail)
- "Follow me on Instagram/TikTok" (handled by channel header)
- Full timestamps section in `[00:00:00]` format (YouTube doesn't need it; `0:00` is the standard)
- Long disclaimer / about-the-channel block (Harrison's channel banner does this)

Keep the description tight. The hook + chapters + hashtags is the whole asset.
