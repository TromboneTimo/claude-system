Best performing content is stored in `references/top-content/best-performing-posts.md`.

**How to apply:** Before writing any social media posts, blog content, or marketing copy, read this file first. Study the tone, structure, and patterns that worked. New social content should match the proven voice:
- Musician-to-musician peer language (chops, embouchure, gig, horn)
- Objection-reframing ("You may think X. We disagree.")
- Practical usage tips woven into the copy
- Real endorser testimonials with specific before/after moments
- Community proof ("Wind Musicians everywhere")
- Direct competitor comparison (wax = bad, wax-free = good)
- "Playing stronger longer" and "Recover faster" as recurring hooks

Add new high-performing posts to this file as they're identified.
