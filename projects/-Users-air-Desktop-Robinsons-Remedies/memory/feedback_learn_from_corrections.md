name: Learn from corrections and don't repeat mistakes
description: When the user corrects an approach, apply that correction to ALL similar situations going forward, not just the one instance
type: feedback
---

When the user corrects something, apply the fix universally and proactively to all similar situations. Do not wait to be told again.

**Why:** The user has had to repeat corrections multiple times (e.g., "flush" not "block" for lactic acid, before/during/after usage, ingredient-first not wax-first messaging, real assets not placeholders). Each repeated correction wastes time and erodes trust.

**How to apply:**
- When corrected on copy tone (e.g., "be fiercer"), apply that tone to EVERY section, not just the one called out
- When corrected on messaging balance (e.g., "lead with ingredients, not just anti-wax"), rebalance the ENTIRE page
- When corrected on asset usage (e.g., "use real images"), apply to ALL endorsers/products across ALL pages
- When corrected on product details (e.g., "it's before/during/after, not just while playing"), update EVERY reference on EVERY page
- Before submitting work, do a self-review: "Would the user have to correct me on this based on past feedback?"
