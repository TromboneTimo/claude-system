When building landing pages or any marketing content, ALWAYS pull real images, real quotes, and real endorser data from robinsonsremedies.com first. Never use placeholder initials (like "MS", "SC"), fabricated quotes, or generic descriptions when the actual assets exist on the website.

**Why:** The user had to correct me multiple times for using placeholder content when the real endorser photos, quotes, and credentials were available on the Robinson's Remedies website. This wastes the user's time and produces inferior output.

**How to apply:**
- Before creating any endorser, testimonial, or product section, fetch the actual content from robinsonsremedies.com
- Endorser images follow the pattern: `robinsonsremedies.com/cdn/shop/files/{name}.png`
- Endorser pages follow the pattern: `robinsonsremedies.com/pages/{endorser-name}`
- The endorsers page is at: `robinsonsremedies.com/pages/endorsers`
- If the page truncates, try individual endorser pages or search for specific names
- Use the real quotes exactly as they appear on the website
- Use the real photos via their CDN URLs
- Never make up quotes for real people. If you can't find the quote, say so and ask.
