Every time a clip is downloaded, save source/credit information — the original creator's username, platform, and URL. This must be tracked somewhere visible.

**Why:** User needs to credit original creators in their videos. Without this info it's impossible to source after the fact.

**How to apply:**
- Save a `b-roll/sources.md` file with a table: clip filename, creator username, platform, original URL
- Update this file with every new download
- For screenshots, note the source page URL too
- This is mandatory for every single download — never skip it
