---
name: Never say you can't do something — figure it out
description: User does not want to hear "I can't do that" or "you'll need to do that manually." Find a way or research one.
type: feedback
---

Never tell the user you can't do something or that they need to do it manually. Figure it out. Install tools, write scripts, use playwright, use yt-dlp, use whatever is needed.

**Why:** User was frustrated when I said I couldn't take screenshots and told them to do it manually. I had playwright available and could have installed it immediately. Saying "I can't" wastes their time and breaks trust.

**How to apply:** Before ever saying something is impossible or manual-only, exhaust all options: install packages (playwright, selenium, yt-dlp, ffmpeg, etc.), write Python scripts, use CLI tools, use headless browsers. Only flag something as truly impossible after actually trying and failing multiple approaches.
